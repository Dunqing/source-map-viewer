# Deep Link to Segment Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `?seg=<index>` query param support so users can link directly to a specific mapping segment.

**Architecture:** Read `seg` param on mount, select the segment and scroll to it. On segment click, update the URL with `replaceState`. Integrate with the existing `syncUrlParams` watcher.

**Tech Stack:** Vue 3, existing VisualizationPage component

### Task 1: Read `?seg` on mount and select segment

**Files:**

- Modify: `src/pages/VisualizationPage.vue`

- [ ] **Step 1: Add seg param reading in onMounted**

In `src/pages/VisualizationPage.vue`, inside the first `onMounted` callback (around line 30), after restoring UI state from URL params, add seg handling:

```ts
// After the existing param restoration (showStats, showMappings, initialTab)
const initialSeg = Number(params.get("seg"));
if (initialSeg >= 0 && initialSeg < store.mappingIndex.length) {
  // Defer to after ready=true so panels are mounted
  const segToSelect = store.mappingIndex[initialSeg];
  requestAnimationFrame(() => {
    requestAnimationFrame(() => {
      store.setHoveredSegment(segToSelect);
      store.setActiveSource(segToSelect.sourceIndex);
      originalPanelRef.value?.scrollToLine(segToSelect.originalLine);
      generatedPanelRef.value?.scrollToLine(segToSelect.generatedLine);
      if (showMappings.value) {
        mappingsPanelRef.value?.scrollToHovered();
      }
    });
  });
}
```

The double `requestAnimationFrame` ensures `ready` is true and panels are fully mounted before scrolling.

### Task 2: Update URL on segment click

**Files:**

- Modify: `src/pages/VisualizationPage.vue`

- [ ] **Step 1: Add seg to syncUrlParams**

In `syncUrlParams` (around line 48), add the `seg` param based on the currently hovered segment:

```ts
function syncUrlParams() {
  if (!urlSyncEnabled) return;
  clearTimeout(urlSyncTimer);
  urlSyncTimer = window.setTimeout(() => {
    const p = new URLSearchParams();
    if (store.activeSourceIndex > 0) p.set("tab", String(store.activeSourceIndex));
    if (showMappings.value) p.set("mappings", "1");
    if (showStats.value) p.set("stats", "1");
    // Add seg param for the clicked segment
    if (store.hoveredSegment) {
      const idx = store.mappingIndex.indexOf(store.hoveredSegment);
      if (idx >= 0) p.set("seg", String(idx));
    }
    const search = p.toString();
    const newUrl = `${window.location.pathname}${search ? "?" + search : ""}`;
    window.history.replaceState(null, "", newUrl);
  }, 16);
}
```
